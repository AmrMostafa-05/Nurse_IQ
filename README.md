# Nurse_IQ

**Description:**  
Nurse_IQ is an educational platform for veterinary and medical students to track courses and skills.  

**Technologies:**  
- ASP.NET Core MVC  
- Entity Framework Core  
- SQL Server  
- Razor Views + Bootstrap  
- Python Chatbot
